Generate badges for GloPress in your projects

## Description

Generate SVG badges of translation statuses in your projects. This plugin requires GlotPress to be installed.

**Badges:**

Project badges:

* <code>translation_sets</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/translation_sets.svg">
* <code>sub_projects</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/sub_projects.svg">

Translation sets badges:

* <code>all_count</code> <img src="https://plugins.svn.wordpress.org/badge-for-glotpress/assets/examples/all-count-fr.svg">
* <code>untranslated_count</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/untranslated_count-fr.svg">
* <code>percent_translated</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/percent_translated-fr.svg">
* <code>waiting_count</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/waiting_count-fr.svg">
* <code>warnings_count</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/warnings_count-fr.svg">
* <code>fuzzy_count</code> <img src="https://ps.w.org/badge-for-glotpress/assets/examples/fuzzy_count-fr.svg">

**Example:**

* https://example.com/glotpress/badge/path/of/the/project/percent_translated-fr.svg

For basic URLs scheme, use this format:

* https://example.com/?gp_badge=true&project_path=the/project/path/&locale=fr&info=percent_translated

## Installation

1. Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress admin
